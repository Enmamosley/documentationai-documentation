# Registro de Entrada y Salida

El sistema mantiene un registro preciso de cuándo ingresa y sale el personal de contratistas.

## Flujo de entrada

1. El contratista se presenta en el punto de acceso con el **PDF del permiso** (impreso o digital)
2. El **guardia** escanea el código QR
3. El guardia verifica visualmente que los técnicos coincidan con las fotos registradas
4. El guardia hace clic en **Registrar Entrada**
5. El sistema registra:
    - Fecha y hora exacta de la entrada
    - Guardia que registró
    - Permiso asociado
6. El estado del permiso cambia a **En Uso**

## Flujo de salida

1. Al terminar el trabajo, el contratista se presenta nuevamente en el punto de acceso
2. El **guardia** escanea el código QR otra vez
3. El guardia hace clic en **Registrar Salida**
4. El sistema registra:
    - Fecha y hora exacta de la salida
    - Guardia que registró
5. El estado del permiso cambia a **Completado**

## Consultar registros

Los registros de entrada y salida son visibles en:

- **Detalles del permiso** — Sección de historial
- **Dashboard del CCO** — Monitoreo en tiempo real
- **Logs de auditoría** — Para administradores

## Consideraciones

<Callout kind="info">
**Tolerancia de horario**

El sistema permite una tolerancia de **30 minutos** después de la hora de entrada autorizada. Pasado ese tiempo, el permiso se **vence automáticamente**.

</Callout>
<Callout kind="alert">
**Sin salida registrada**

Si el guardia no registra la salida y la hora autorizada ya expiro, el sistema puede cerrar el permiso automaticamente y marcarlo como **Completado** con una salida registrada por sistema.

</Callout>
