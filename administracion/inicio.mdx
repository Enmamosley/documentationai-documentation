# Administración

Esta sección es para el personal interno de Pabellon M. Cubre monitoreo, revision de permisos, configuracion del sistema y control operativo.

## Roles en esta sección

| Rol | Funcion principal |
|---|---|
| **Administrador** | Control total — usuarios, empresas, workflows, auditoria |
| **Atencion al Cliente** | Revision, clasificacion y aprobacion de permisos |
| **Mantenimiento** | Revision tecnica + creacion propia con auto-aprobacion |
| **Operaciones** | Revision tecnica departamental |
| **Seguridad** | Revision tecnica departamental |
| **Pabellon M** | Creacion de permisos con flujo departamental directo |
| **Sistemas Especiales** | Creacion de permisos con flujo departamental directo |
| **CCO** | Monitoreo en tiempo real y control de acceso |
| **Guardia** | Escaneo de QR y registro de entrada y salida |
