# Notificaciones por Email

Además de las notificaciones dentro del sistema, se envían **correos electrónicos** para los eventos más importantes.

## ¿Cuándo se envía un email?

| Evento | Contenido del email |
|---|---|
| **Permiso aprobado** | Confirmación + enlace para descargar el PDF con QR |
| **Permiso rechazado** | Motivo del rechazo y observaciones |

## Enlace de descarga seguro

Cuando tu permiso es aprobado, el email incluye un **enlace directo** para descargar el PDF:

- No necesitas iniciar sesión para usarlo
- El enlace contiene un token de seguridad
- Es de un solo uso o de tiempo limitado

## WhatsApp (si está configurado)

El sistema también puede enviar notificaciones por **WhatsApp** para los eventos de permiso aprobado.

<Callout kind="info">
**Configuración de WhatsApp**

La integración con WhatsApp depende de la configuración del sistema. Consulta con el administrador si esta funcionalidad está activa.

</Callout>
## Problemas con notificaciones

Si no recibes notificaciones por email:

1. Verifica que tu correo en el sistema sea correcto (en **Mi Perfil**)
2. Revisa la carpeta de **spam** o correo no deseado
3. Contacta al administrador del sistema
