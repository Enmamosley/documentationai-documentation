# Notificaciones del Sistema

El sistema envía notificaciones para mantenerte informado sobre el estado de los permisos y otras actividades relevantes.

## Tipos de notificaciones

### Notificaciones dentro del sistema

- Aparecen en el **icono de campana** :material-bell: en la barra superior
- Incluyen un **contador** con las notificaciones no leídas
- Haz clic en la campana para ver el listado

### Ver todas las notificaciones

1. Haz clic en el icono de **campana**
2. Selecciona **Ver todas** para acceder al historial completo

## Eventos que generan notificaciones

| Evento | ¿Quién la recibe? |
|---|---|
| Nuevo permiso creado | Atención al Cliente |
| Permiso escalado a departamento | Departamento correspondiente |
| Permiso aprobado | Contacto solicitante |
| Permiso rechazado | Contacto solicitante |
| Entrada registrada | CCO |
| Salida registrada | CCO |
| Permiso vencido | Contacto solicitante |
| Permiso cancelado automáticamente | Contacto solicitante |

## Marcar como leída

- Haz clic en la notificación para marcarla como leída y ver los detalles
- Usa **Marcar todas como leídas** para limpiar el contador
