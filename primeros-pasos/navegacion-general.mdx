# Navegación General

## Estructura de la interfaz

El sistema tiene una interfaz consistente con los siguientes elementos:

### Barra superior

- **Logo de Pabellón M** — Haz clic para volver al Dashboard
- **Menú de navegación** — Acceso rápido a las secciones principales
- **Notificaciones** — Icono de campana con contador de notificaciones pendientes
- **Tu perfil** — Tu nombre con menú desplegable (perfil, cerrar sesión)

### Menú lateral

Dependiendo de tu rol, verás diferentes opciones en el menú:

| Sección | Descripción |
|---|---|
| **Dashboard** | Vista general con estadísticas y accesos rápidos |
| **Permisos** | Crear, ver y gestionar permisos de trabajo |
| **Empresas** | Gestión de empresas (solo Admin) |
| **Usuarios** | Gestión de usuarios (solo Admin) |
| **Notificaciones** | Historial de notificaciones |

### Área de contenido

El contenido principal se muestra en el centro de la pantalla. Aquí verás formularios, tablas, dashboards y demás información.

## Tablas y listados

Las tablas del sistema incluyen:

- **Buscador** — Filtra resultados por texto
- **Paginación** — Navega entre páginas de resultados (15 elementos por página)
- **Acciones** — Botones para ver, editar o eliminar registros

## Navegación con el teclado

- Usa ++tab++ para moverte entre campos de formulario
- Usa ++enter++ para enviar formularios
- Usa los botones de navegación del navegador para volver atrás
